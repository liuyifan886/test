# 几个常用的git指令
git add .（意思就是把你的这个文件的东西放在一辆车上）
git commit -m “xxx”（把这辆车上的东西给拉到仓库里面去，xxx 代表你对于这个车上的东西的说明）
git push（把你本地的仓库的东西给推送到远程的网站上面去）、
git pull（把远程的网站上面的东西拉到你的本地工作室）
