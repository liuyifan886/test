和java的对象一样
是一种无序的数据集合
用来表述某个事物

声明语法
let 对象名={ }
let 对象名=new Object()
实际开发中，多用花括号那个方法

对象有属性和方法组成
let 对象名={
属性名:属性值，
方法名:函数
}


增删改查
查：第一种写法：对象名.属性名  第二种写法：对象名\['属性名']
改：重新做一个赋值操作 对象名.属性名=新值
增：语法和改一样
删(最新的语法不允许这么删)： delete 对象名.属性名

**查的第二种方法会大量使用，记得加单引号**
